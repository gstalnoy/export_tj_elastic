Обработки тестирования

Автотестирование себестоимости

Обработка предназначена для тестирования конфигурации "1С: ERP Управление предприятием 2"

Выполняет перерасчет себестоимости и сравнивает с ранее расчитанной. Произвести перерасчет можно либо с начиная с указанного месяца, либо с месяца в котором есть первые данные о себестоимости.

Выполнить обновление ИБ

Предназначена для тестирования конфигураций содержащих БСП

Выполняет обработчики обновления. В разделенной базе сначала выполняются монопольные обработчики в нулевой области, далее - последовательно, обрабочики в каждой из областей. В зависимости от настройки в обработке "Детальные ошибки по каждому обработчику" - формирует либо одну общую ошибку, либо отдельные ошибки по каждому невыполненному обработчику

Открытие всех форм

Предназначена для тестирования конфигураций содержащих БСП

Тестирует открытие форм справочников и документов. Умеет открывать формы элементов по ссылке. Проверяет что при открытии форма не падает с исключением.

Перезапись справочников и ПВХ

Предназначена для тестирования конфигураций содержащих БСП

Перезапись объектов справочников и планов видов характеристик. Проверяет корректность заполнения объектов.

В настройках можно можно перечислить ссылки исключаемые из тестирования.

Перепроведение документов и сравнение движений

Обработка предназначена для тестирования конфигурации "1С: ERP Управление предприятием 2"

Перепроводит документы и сравнивает сформированные движения регистров до и после проведения. Расхождения отображаются в отчете. При перепроведении проверяет корректность заполнения.

Проверка консистентности регистров

Предназначена для тестирования конфигураций содержащих БСП

Проверяет что существующие наборы записей в регистрах: не содержат записей с пустыми измерениями и выставленным флагом "Запрет незаполненных значений" проходят проверку заполнения

При наличии проблемных записей выводится отчет

Сверка регистров до и после обновления

Предназначена для тестирования конфигураций содержащих БСП

Выгружает записи регистров и сохраняет в каталог. После обновления ИБ, записи регистров выгружаются повторно и сравниваются с эталонными (до обновления). Все найденные различия выводятся в отчет.

Тестирование печатных форм

Предназначена для тестирования конфигураций содержащих БСП

Формируются печатные формы с настройками по умолчанию и сохраняются в каталог. После обновления ИБ, печатные формы формируются повторно и сравниваются с эталонными (до обновления). На все найденные различия формируются ошибки.

Тест Коммандного интерфейса

Предназначена для тестирования конфигураций содержащих БСП

Тестируемое приложение запускается в режиме ТестМенеджера. Для выбранных пользователей с ограниченными правами запускается приложение в режиме ТестКлиента и "прокликиваются" все команды доступные в глобальном командном интерфейсе. При возникновении на ТестКлиенте исключении - формируется ошибка.

Формирование и сравнение отчетов

Предназначена для тестирования конфигураций содержащих БСП

Формируются отчеты с настройками по умолчанию и сохраняются в каталог. После обновления ИБ, отчеты формируются повторно и сравниваются с эталонными (до обновления). На все найденные различия формируются ошибки.
